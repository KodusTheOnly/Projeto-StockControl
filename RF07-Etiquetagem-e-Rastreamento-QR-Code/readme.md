* para fazer um qr code, precisa criar um código "específico" para o qrcode. 
# processo de criação
Por exemplo, o produto é um presunto, e quero criar um qrcode de validade para ele. Vejo com o fabricante a validade, e então, anoto essa validade em um site, onde eu anoto o nome do produto, validade, quantidade, (adicionar mais caso seja necessário), e depois, ao criar estas informações, é gerado um qrcode, esse qrcode leva para um site que avisa a data, quantos dias falta para vencer, e vai ter a opção de excluir ele do banco de dados, caso tenha expirado.

nesse site, no formato mobile vai mostrar todos os produtos e a validade deles, (e pode escolher a data de aviso para estes produtos, como um alarme(lembrete). pode excluir ou adicionar produtos.) DECIDIR COM A ARLETE, MESMO RF

Os dados de excluir e adicionar o produto precisa ser registrado em algum lugar para comunicar com o resto das RFs, para fazer a gestão e relatório.
